# 🚚 Tata Supply Chain CO LTD - DBMS Project

## 📌 Overview

A Database Management System designed for Tata Supply Chain CO LTD to streamline procurement, inventory management, vendor management, quality inspection, and material issue processes.

## 🎯 Objectives

* Manage Parts and Inventory
* Maintain Vendor Records
* Track Purchase Orders and Challans
* Generate Goods Received Reports (GRR)
* Record Quality Inspection Results
* Manage Material Issue Requisitions (MIR)

## 🛠️ Technologies Used

* MySQL
* MySQL Workbench
* Draw.io
* GitHub

## 📂 Project Modules

* Part Category Management
* Inventory Management
* Vendor Management
* Purchase Orders
* Transporters & Challans
* GRR Management
* Quality Testing
* MIR Management
